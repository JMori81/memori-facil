## Instalação
1. Clone o repositório:
   ```bash
   git clone https://github.com/JMori81/memori-facil.git
   cd memori-facil
   ```
2. Instale as dependências:
   ```bash
   npm install
   # ou
   yarn install
   ```

## Rodando em desenvolvimento
```bash
npm run dev
# ou
yarn dev
```

## Build / Produção
